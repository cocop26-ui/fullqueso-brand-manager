## 1. IDENTIDAD DE MARCA

### 1.1 Quiénes somos

Full Queso es una franquicia venezolana de comida rápida artesanal especializada en **tequeños, churros y helados**, con presencia en múltiples locales en Caracas (Chacao, Las Mercedes, El Hatillo, Candelaria, El Marqués, La Castellana).

Somos la marca que convirtió el tequeño venezolano en una experiencia de calle premium: rápida, accesible, irresistible y 100% venezolana.

### 1.2 Misión

Llevar el sabor y la alegría de compartir venezolano a cada momento del día, desde la lonchera de los niños hasta el antojo de medianoche.

### 1.3 Propuesta de valor

- **Producto artesanal con velocidad de fast food**
- **Precio accesible** (punto de quiebre probado: $15 USD)
- **Identidad cultural venezolana auténtica**
- **Experiencia de compartir**, no de consumo solitario

### 1.4 Territorio de marca

**COMPARTIR**

Todo lo que hace Full Queso invita a compartir: los productos vienen en porciones para grupos, el lenguaje es de "pana a pana", los combos están pensados para el colectivo.

### 2.1 Arquetipo: EL BUFÓN VENEZOLANO

| Atributo | Descripción |
|----------|-------------|
| **Arquetipo** | El Bufón / The Jester |
| **Territorio** | COMPARTIR |
| **Personalidad** | Mundo de calle, slang cotidiano, pana de toda la vida, aspiracional pero real |
| **Tono** | "Echando broma pero en serio" |
| **Energía** | Alegre, irreverente, cálido, directo |

### 2.2 Voz de marca — DO

- Habla en venezolano cotidiano: "pana", "chamo", "marico", "causa", "verga qué bueno"
- Usa primera persona colectiva: "nosotros los venezolanos", "aquí en Venezuela"
- Sé directo y confiante. No pidas permiso para ser bueno
- Usa el humor como herramienta de conexión, no de distracción
- Mezcla energía y calidad: puedes ser gracioso Y hablar de lo bueno que es el producto
- Usa emojis con criterio — son parte del lenguaje, no decoración

### 2.3 Voz de marca — DON'T

- No seas formal ni corporativo
- No uses tecnicismos de marketing ("engagement", "reach", "KPI") en copy público
- No seas apologético ni inseguro
- No uses el humor que reste al producto
- No uses español neutro cuando puedes usar venezolano
- No finjas ser lo que no eres: somos venezolanos, de Caracas, de la calle

### 2.4 Ejemplos de tono

**Correcto:**
> "8 Nuggets + 5 Full Quesos. Porque el hambre no avisa, pana. 🧀🔥"

**Incorrecto:**
> "Disfruta nuestra deliciosa combinación de nuggets y tequeños artesanales."

**Correcto:**
> "¿Ya probaste el Mini Placer? Nuggets + Churros + Sundae. Sí, todo eso junto. Te lo mereces."

**Incorrecto:**
> "Prueba nuestro nuevo combo que incluye nuggets, churritos y helado."

## 4. IDENTIDAD VISUAL

### 4.1 Colores — Paleta oficial

#### Color 1: Full Queso Red (PRIMARIO)
```
HEX:     #FF0000
RGB:     255, 0, 0
CMYK:    0, 100, 100, 0
Pantone: Red 032 C
```
**Uso:** Fondos de logo, ribbons, badges, banners, empaques, CTAs. Es el color dominante y dueño de la marca. Nunca modificar.

#### Color 2: Pure White
```
HEX:     #FFFFFF
RGB:     255, 255, 255
```
**Uso:** Tipografía sobre fondo rojo. Logo en versión negativa. Espacios de respiro.

#### Color 3: Cheese Yellow (SECUNDARIO)
```
HEX:     #FFD700
RGB:     255, 215, 0
CMYK:    0, 16, 100, 0
Pantone: 116 C
```
**Uso:** Fondos secundarios (menuboard, piezas publicitarias, wallmenu). Comunica energía y asociación directa al queso.

#### Color 4: Deep Dark (ACENTO)
```
HEX:     #1A1A1A
RGB:     26, 26, 26
Pantone: Black 6 C
```
**Uso:** Labels descriptivos de productos sobre fondos claros. Texto secundario. Nunca como fondo principal.

### 4.2 Reglas de color

- **Combinación dominante:** Texto blanco sobre fondo rojo
- **Combinación secundaria:** Elementos rojos y blancos sobre fondo amarillo
- **Prohibido:** Gradientes, transparencias, variaciones de los colores oficiales
- **Monocromático:** Solo versiones todo-blanco o todo-negro para aplicaciones de un solo color

### 4.3 Proporción de uso de color en piezas

| Color | Proporción recomendada |
|-------|----------------------|
| Full Queso Red | 50–60% |
| Cheese Yellow | 20–30% |
| Pure White | 15–25% |
| Deep Dark | 5–10% |

## 5. TIPOGRAFÍA

### 5.1 Sistema tipográfico — 5 niveles

#### N1 — Display Hero (Script)
```
Fuente:      Script cursiva — referencia visual: Pacifico / Great Vibes
Pesos:       Regular + Bold
Uso:         Nombres de combos en ribbons publicitarios
Tamaño ref:  72–96px digital / 60pt+ print
Ejemplos:    "Nuggets + Tequeños", "Mini Placer"
```

#### N2 — Subtítulo Bold
```
Fuente:      Sans-serif bold — referencia: Inter Bold / Gotham Bold
Pesos:       Bold 700 / ExtraBold 800
Uso:         Complemento al N1. Subtítulos de combo.
Tamaño ref:  48–64px digital / 36pt+ print
Ejemplos:    "+ Tequeños", "+ Mini Placer"
```

#### N3 — Headlines UI (OFICIAL)
```
Fuente:      Inter — FUENTE OFICIAL (Brand Guidelines)
Pesos:       Bold 700 / SemiBold 600
Uso:         Titulares en piezas digitales, web, apps
Descarga:    https://fonts.google.com/specimen/Inter
Tamaño ref:  28–48px digital / 18–36pt print
```

#### N4 — Body & UI (OFICIAL)
```
Fuente:      Inter — FUENTE OFICIAL (Brand Guidelines)
Pesos:       Regular 400 / Medium 500
Uso:         Cuerpo de texto, botones, captions, interfaces, etiquetas
Descarga:    https://fonts.google.com/specimen/Inter
Tamaño ref:  14–20px digital / 10–14pt print
```

#### N5 — Logo / Empaque
```
Fuente:      Condensed Bold — referencia: Bebas Neue / Impact
Estilo:      All caps, condensed, bold
Uso:         "FULL QUESO", "NUGGETS", "TEQUEÑOS", "CHURROS" en packaging
Nota:        Confirmar fuente exacta con archivo original .ai/.psd del diseñador
```

#### Badges y Tags
```
Fuente:      Sans-serif ExtraBold / Heavy — all caps
Uso:         "NUEVOS", etiquetas promocionales, pill badges
Presentación: Fondo rojo redondeado (#FF0000), texto blanco, sin borde
```

### 5.2 Fuentes descargables recomendadas (Google Fonts)

| Fuente | Rol | URL |
|--------|-----|-----|
| Inter | Headlines + Body (OFICIAL) | fonts.google.com/specimen/Inter |
| Pacifico | Script Display N1 (referencia) | fonts.google.com/specimen/Pacifico |
| Great Vibes | Script Display N1 (alt. referencia) | fonts.google.com/specimen/Great+Vibes |
| Bebas Neue | Condensed N5 (referencia) | fonts.google.com/specimen/Bebas+Neue |
| Montserrat | Alt. sans-serif titulares | fonts.google.com/specimen/Montserrat |

### 5.3 Reglas tipográficas

- **Nunca** cambiar la tipografía del logo
- **Nunca** estirar, distorsionar ni alterar el interletraje
- **Máximo 2 familias** tipográficas por pieza
- Los badges y labels de empaque siempre en **ALL CAPS**
- Inter es la fuente por defecto para **todo contenido digital**

## 6. LOGO

### 6.1 Variantes aprobadas

| Variante | Layout | Uso | Tamaño mínimo |
|----------|--------|-----|---------------|
| **Principal** | Horizontal: FULL QUESO + cheese wheel en Q | Comunicaciones principales, web, tarjetas | 120px / 1.5in |
| **Secundario** | Stacked: FULL encima de QUESO + cheese wheel | Redes sociales cuadradas, apps, espacio reducido | 100×100px / 1in |
| **Ícono** | Cheese wheel standalone en círculo | Favicon, avatar, watermark | 32px / 0.5in |
| **Outlined** | Logo con outline blanco | Sobre fondos complejos o fotografía | Variable |

### 6.2 Clear space

El espacio mínimo libre alrededor del logo en todos sus lados = altura de la letra **"L"** en "FULL".
- Digital: 20px mínimo
- Print: 0.25 pulgadas mínimo

### 6.3 Logo — DO

- Logo blanco sobre fondo Full Queso Red ✓
- Logo rojo sobre fondo blanco ✓
- Logo blanco con outline sobre fotografía ✓
- Logo negro sobre fondo blanco (solo aplicaciones monocromáticas) ✓

### 6.4 Logo — DON'T

- No rotar el logo a ningún ángulo ✗
- No usar sobre fondos sin suficiente contraste ✗
- No aplicar sombras, gradientes ni efectos ✗
- No separar el cheese wheel del texto ✗
- No cambiar colores del logo ✗
- No usar archivos de baja resolución ✗

## 7. FOTOGRAFÍA Y VIDEO

### 7.1 Estilo fotográfico

- **Sujeto:** El producto es siempre el héroe de la imagen
- **Luz:** Natural o ring light. Rica, cálida, sin sombras duras
- **Color:** Saturado, vibrante, cálido. Siempre en línea con la paleta de marca
- **Composición:** Clean, close-up que destaque textura y calidad
- **Fondo:** Cheese Yellow (#FFD700), superficies rojas, neutros cálidos
- **Evitar:** Ambientes fríos, azules, fondos oscuros sin producto en primer plano

## 9. APLICACIONES DE MARCA EN SOPORTES

### 9.1 Empaques (Cajas / Bolsas)

- Fondo: Full Queso Red (#FF0000)
- Logo: Variante Stacked, blanco
- Tipografía de soporte: Condensed Bold All Caps, blanco
- Descriptor bajo logo: Nombre del producto (NUGGETS, TEQUEÑOS, CHURROS) en tamaño reducido
- Elemento decorativo: Rombo/diamante negro en esquina inferior

### 9.2 Menuboard / Wallmenu

- Fondo general: Cheese Yellow (#FFD700)
- Ribbons de combo: Fondo rojo redondeado (#FF0000)
- Nombre de combo N1: Script cursiva blanca sobre ribbon
- Subtítulo N2: Sans-serif bold blanco sobre ribbon
- Labels de ítems N3: Sans-serif bold dark (#1A1A1A), flotante sin caja
- Badge "NUEVOS": Pill rojo, texto blanco all-caps, esquina superior
- Elemento de cierre: Diamante negro esquina inferior derecha

### 9.3 Piezas digitales (Instagram / Meta Ads)

- Ratio 9:16 para stories y combos de lanzamiento
- Ratio 4:5 para feed
- Fondo dominante: Cheese Yellow o Full Queso Red según campaña
- Logo: Watermark sutil esquina inferior derecha
- Tipografía de overlay: Inter Bold para subtítulos, Inter Regular para copy
- CTA: Siempre en pill/badge rojo con texto blanco

## 10. REGLAS ABSOLUTAS — DO & DON'T

### ✅ SIEMPRE

1. Usar la paleta de colores oficial sin modificaciones
2. Usar Inter como fuente base en todo contenido digital
3. Producto como héroe visual en toda pieza fotográfica
4. Respetar el clear space del logo
5. Hablar en venezolano auténtico, no español neutro
6. Mantener All Caps en badges, empaques y labels
7. Confirmar fuentes display con archivos originales antes de producir

### ❌ NUNCA

1. Modificar los colores de marca (ni siquiera "un poco más oscuro")
2. Aplicar gradientes, transparencias ni efectos al logo o colores
3. Rotar, distorsionar o estirar el logo
4. Usar más de 2 familias tipográficas en una sola pieza
5. Hablar en tono corporativo o formal
6. Usar imágenes de baja resolución del logo o productos
7. Colocar el logo sobre fondos sin contraste suficiente
8. Separar el cheese wheel del logotipo
